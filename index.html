Animal Pose Estimation

A deep learning project for estimating animal poses from images and video.

View the project on GitHub

About the project

This project explores deep learning techniques for detecting and estimating animal body poses.

Project goals

Our goal is to develop a model that can identify key points on an animal's body and use them to study its posture and movement.

Methods

The project uses computer vision, pose-estimation models, and annotated animal-image datasets.
